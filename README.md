# ma2 to simai automatic
该项目只是为了更方便转谱

## 使用说明
- **ChartData**: 存放simai.txt的文件夹 `.txt`
- **CoverData**: 存放乐曲封面的文件夹 `.png`
- **SoundData**: 存放音频的文件夹 `.awb`
- **MovieData**: 存放视频的文件夹 `.dat`

## 前置
```bash
pip install -r requirements.txt
```

## 其他说明
- **python版本**: 3.9以上
- **NET版本**: 8.0
